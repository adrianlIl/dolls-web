一個使用 Nuxt 4 開發的公仔網站。

## 快速開始

安裝依賴套件：
```bash
pnpm install
```

啟動開發伺服器：
```bash
pnpm dev
```

在瀏覽器中開啟 [http://localhost:3000](http://localhost:3000)。

## 建置生產版本

```bash
pnpm build
pnpm preview
```
